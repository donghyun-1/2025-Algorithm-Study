## 📌 스터디 규칙

#### 진행 방식

1. 각자 번갈아 가면서 그날 풀 문제를 캘린더에 추가한다.
   
2. 문제를 풀어서 인증샷을 캘린더에 올린다. (**새벽 6시까지** →solved.ac 기준)
   
3. 월 수 주말(토or일)에 디스코드 채널에서 각자 문제를 어떤 식으로 풀었고 대화 해보는 시간을 가진다.
